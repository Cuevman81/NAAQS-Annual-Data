# NAAQS-Annual-Data

Readme

Simple R scripts using [EPA Pre Generated Data Files](https://aqs.epa.gov/aqsweb/airdata/download_files.html), [EPA AQS API](https://aqs.epa.gov/aqsweb/documents/data_api.html) and [aqsr](https://github.com/jpkeller/aqsr) to find site statistics, trends, and annual max(es) for NAAQS criteria pollutants adding simple plots of data across the state of Mississippi
